# 《V5.0 底层数据库：高净值人群资产画像字段定义（绝密版）》

**文档编号：** V5-CORE-DATA-2026-SECRET **加密等级：** 零知识加密（只有协议拥有者可全局透视）

## 一、 资产主体字段 (The Asset DNA)

_这些字段用于定义“物”，确立资产的唯一性。_

1. **`Asset_Global_UID`**: 全球唯一资产识别码（基于母子纠缠算法生成的哈希）。
    
2. **`Birth_Certificate_ID`**: 生产端激活 ID。记录该资产由哪个工厂、哪张母卡在何时激活。
    
3. **`Commodity_Category`**: 资产类目。如：奢侈品包、腕表、名酒、稀有金属等。
    
4. **`Primary_Market_Price`**: 首次激活时的官方指导价（用于计算资产残值）。
    
5. **`Embedded_Chip_Type`**: 物理芯片规格（如 NTAG 424 DNA）。
    

## 二、 持有人动态画像 (The Holder Profile)

_这是最值钱的部分，用于定义“人”，构建高净值人群地图。_

1. **`Current_Owner_Tier`**: 持有人等级。
    
    - _Tier 0_: 品牌方（未激活）。
        
    - _Tier 1_: 初次购买者。
        
    - _Tier 2-N_: 二手/多次流转持有人。
        
2. **`Asset_Density_Index`**: 资产密度指数。该用户 ID 下挂载的所有 V5.0 认证资产的总价值。
    
    - _意义：_ 识别他是“入门级”还是“核心收藏家”。
        
3. **`Holding_Duration_Avg`**: 平均持有周期。记录资产在该用户手中停留的时间。
    
    - _意义：_ 区分“自用客户”与“投机炒客（黄牛）”。
        
4. **`Geographic_Trajectory`**: 地理迁徙轨迹。每次扫码（验证/过户）时的经纬度与城市代码。
    
    - _意义：_ 监控资产在全球范围内的流动方向（如：资产正在从东亚流向中东）。
        

## 三、 交易行为特征 (The Transaction Logic)

_用于捕捉“钱”的流动模式。_

1. **`Clearing_Event_Log`**: 清算事件记录。每一笔 200 元确权费的支付流水与分账时间戳。
    
2. **`Appreciation_Depreciation_Rate`**: 资产溢价/折价率。对比前一次过户与本次过户的成交价（需选填）。
    
    - _意义：_ 实时反映该品牌在二级市场的保值率。
        
3. **`Authentication_Frequency`**: 验证频率。该资产被扫码验证但未发起过户的次数。
    
    - _意义：_ 高频验证意味着该资产正在市场上被多个潜在买家“询价”。
        

## 四、 社交与关联关系 (The Social Topology)

_用于定义“圈子”。_

1. **`Circle_Transfer_Link`**: 圈层流转链。记录 A 经常过户给 B，B 经常过户给 C。
    
    - _意义：_ 自动勾勒出高净值社交圈（谁是这个圈子的意见领袖）。
        
2. **`Cross_Brand_Affinity`**: 跨品牌关联度。
    
    - _意义：_ 发现“买爱马仕的人同时也买了哪些品牌”，这是卖给 Meta/Google 最核心的交叉销售数据。
