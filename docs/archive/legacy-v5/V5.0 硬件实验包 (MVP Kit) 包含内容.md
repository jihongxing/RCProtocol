## V5.0 硬件实验包 (MVP Kit) 包含内容

为了跑通上述逻辑，你的“实验包”需要包含以下物理组件：

### 1. 核心芯片 (Tags)

- **NTAG 424 DNA 湿嵌体 (子卡)** x 20 枚：用于贴在产品上。
    
- **NTAG 424 DNA 高级 PVC 卡 (母卡)** x 3 张：建议选择黑色哑光或金属拉丝质感，用于模拟“主权母卡”。
    

### 2. 读写设备 (Hardware)

- **ACR122U NFC 读写器** x 1 台：连接电脑进行底层 APDU 指令调试和 KMS 逻辑联调。
    
- **支持 NFC 的 Android 手机** x 1 部：用于安装自研点睛 App 演示工具。
    
- **iPhone (iOS 13+)**：用于演示“免 App、一碰即显 H5”的极致用户体验。
    

### 3. 开发授权 (Software & Docs)

- **KMS 测试环境接入点**：我为你设计的密钥衍生算法（KDF）及后端校验接口。
    
- **SDM 配置脚本**：用于一键将空白芯片配置为“动态 URL 镜像模式”。
    
- **H5 演示模板**：包含“真伪校验”、“行情看板”和“模拟支付确权”三个环节。
