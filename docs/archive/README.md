# 历史归档说明

`archive/` 用于保存 RCProtocol 在建立统一文档源之前的历史文档、专题草稿、愿景白皮书、讨论记录与被替代版本。

## 规则

- 本目录内容 **不是当前实现依据**
- 如与 `docs/index.md` 及其引用的新体系冲突，一律以新体系为准
- 本目录的价值在于保留历史思考、演进过程与原始素材

## 当前归档分层

- `archive/legacy-v5/`：早期 V5 与概念设计材料
- `archive/legacy-ssot-migration/`：本次统一文档源改造前的旧主文档、专题文档与整理版

后续若需要保留旧稿，应优先归入对应归档目录，而不是重新放回顶层文档区。
